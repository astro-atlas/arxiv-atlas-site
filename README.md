# ArXiv Atlas - Galaxy Evolution Knowledge Portal

A living wiki that tracks, summarizes, and connects galaxy evolution research from ArXiv.

## Project Structure

```
arxiv-atlas/
├── wiki/           # HTML pages for concepts and papers
├── data/           # Paper metadata, network graph, DOIs
├── scripts/        # Processing scripts
├── logs/           # Daily processing logs
├── STATUS.md       # Current project status
└── README.md       # This file
```

## Core Features

1. **Daily ArXiv Processing**: Automatically reads new astro-ph.GA papers
2. **Concept Wiki**: Wikipedia-style pages tracking claims and debates
3. **Paper Network**: Visual graph of paper relationships
4. **Smart Alerts**: Notifications for breakthrough/morphology papers
5. **Daily Digests**: Summaries highlighting agreements/contradictions

## Access

Once running, the wiki will be available at: http://localhost:[PORT]
